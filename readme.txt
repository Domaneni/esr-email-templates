Enhance your Easy School Registration emails with custom templates for every wave.

== Description ==

Email Templates is a module for Easy School Registration plugin you can find for free here too. is a tool to help you manage your school registrations.

It allows you to create custom email templates and assign them to specific waves. Now you are not stuck with one version of email template is settings.

= Features =
* Simple & Intuitive
* Custom email for each wave
* No limits for templates

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "ESR Email Templates"
3. **Activate** ESR Email Templates from your Plugins page

== Frequently Asked Questions ==

= Do I need another plugin for this to run? =

**YES** You need main plugin called Easy School Registration.
